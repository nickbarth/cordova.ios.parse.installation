## Cordova iOS Install

## About this Plugin

Setup device installation in iOS for current user.

## Using the Plugin

Example:

```
window.iOSInstall(function () {
  console.log('Install Setup For Current User');
});
```

## Adding the Plugin ##

```
  cordova plugin add https://github.com/nickbarth/cordova.ios.parse.installation.git
```
